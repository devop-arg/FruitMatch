# Changelog

All notable changes to FruitMatch will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-16

### Added
- Initial public release
- Three game modes: Classic, Adventure, and Zen
- Match-3 gameplay with horizontal and vertical matches
- 2x2 square match detection for bonus points
- Recursive match propagation system
- Multi-language support (English and Spanish)
- Automatic browser language detection
- Global leaderboard system
- Admin panel for score management
- Docker support for API deployment
- Responsive design for all devices
- Sound effects and background music
- Multiple visual themes
- Power-up system
- Combo multiplier mechanics

### Features
- Clean, modern UI with smooth animations
- Offline gameplay support
- Persistent score storage via REST API
- Mobile-friendly touch controls
